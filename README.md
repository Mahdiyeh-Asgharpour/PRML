This repository contains practical assignments developed for an MSc-level Pattern Recognition and Machine Learning (PRML) course. The notebooks cover a custom implementation of Principal Component Analysis (PCA) for color-image transformation and reconstruction, Gaussian modeling of grayscale and RGB image regions, KL-divergence-based similarity and homogeneity analysis, and mutual-information-based image registration using exhaustive translation search. The repository also explores Bayesian decision theory through MAP and minimum-risk classification thresholds, as well as linear regression using polynomial and Gaussian basis functions, least-squares estimation, MAP estimation, regularization, and train/test error analysis. The main objective is to connect the mathematical foundations of statistical pattern recognition with reproducible numerical experiments, visualizations, and practical Python implementations.
